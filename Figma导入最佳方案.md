# Figma导入最佳方案

## 🎯 问题：导入后显示为黑色

**原因**：HTML文件依赖外部CDN资源（Tailwind CSS、Font Awesome），Figma无法加载这些资源。

## ✅ 解决方案（按推荐顺序）

### 方案1：截图 + 手动重建（最推荐）⭐

**优点**：
- ✅ 完全可编辑
- ✅ 所有元素都是Figma原生对象
- ✅ 可以创建组件和设计系统
- ✅ 性能最好

**步骤**：

1. **在浏览器中打开界面**
   ```
   启动服务器 → 打开 http://127.0.0.1:8000/home.html
   ```

2. **设置浏览器视图**
   - 按F12打开开发者工具
   - 点击设备工具栏图标（或按Ctrl+Shift+M）
   - 选择iPhone 15 Pro（393×852px）
   - 确保界面完整显示

3. **截图**
   - 使用截图工具（Snipaste、微信截图、Windows截图工具）
   - 截取整个界面（393×852px）

4. **在Figma中导入**
   - 创建新Frame：393×852px
   - 粘贴截图（Ctrl+V）作为参考层
   - 锁定参考层（右键 → Lock）

5. **手动重建**
   - 参考截图，使用Figma工具重新创建所有元素
   - 使用Auto Layout组织布局
   - 创建Component（按钮、卡片等）

### 方案2：使用优化后的HTML文件

我已经创建了 `home-figma-optimized.html` 作为示例：

**特点**：
- ✅ 移除了所有外部CDN依赖
- ✅ 所有样式内联
- ✅ 使用Emoji替代图标（避免外部资源）
- ✅ 简化了CSS（移除backdrop-filter）

**使用方法**：
1. 在Figma中安装 `html.to.design` 插件
2. 选择 "Import from File"
3. 选择 `home-figma-optimized.html`
4. 导入后检查效果

**注意**：如果还是显示黑色，说明插件无法正确解析HTML，建议使用方案1。

### 方案3：使用Figma的HTML导入插件（需要网络）

**前提条件**：
- 必须启动本地服务器
- 插件需要能访问 `http://127.0.0.1:8000`

**步骤**：
1. 启动服务器（双击 `start-server.bat`）
2. 在浏览器中测试URL是否可访问
3. 在Figma插件中粘贴URL
4. 如果还是黑色，说明插件无法加载外部资源

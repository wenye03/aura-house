# 启动本地服务器指南

由于Figma插件无法直接访问本地文件，需要先启动本地服务器。

## 方法一：使用Python（最简单，推荐）

### Windows/Mac/Linux通用

1. **打开终端/命令提示符**
   - Windows: 按 `Win + R`，输入 `cmd` 或 `powershell`
   - Mac: 打开"终端"应用
   - Linux: 打开终端

2. **进入项目目录**
   ```bash
   cd "E:\STUDY\Works\毕设"
   ```

3. **启动服务器**
   ```bash
   # Python 3
   python -m http.server 8000
   
   # 或如果python命令不存在，尝试
   python3 -m http.server 8000
   ```

4. **访问页面**
   - 打开浏览器访问：`http://localhost:8000/prototype-index.html`
   - 或访问：`http://127.0.0.1:8000/prototype-index.html`

5. **在Figma插件中使用**
   - 在html.to.design插件中输入：`http://localhost:8000/prototype-index.html`

## 方法二：使用Node.js的http-server

### 安装Node.js（如果还没有）
1. 访问 https://nodejs.org/
2. 下载并安装LTS版本

### 安装http-server
```bash
npm install -g http-server
```

### 启动服务器
```bash
# 进入项目目录
cd "E:\STUDY\Works\毕设"

# 启动服务器
http-server -p 8000
```

### 访问页面
- 浏览器访问：`http://localhost:8000/prototype-index.html`

## 方法三：使用VS Code的Live Server插件

### 安装VS Code（如果还没有）
1. 访问 https://code.visualstudio.com/
2. 下载并安装

### 安装Live Server插件
1. 打开VS Code
2. 点击左侧扩展图标（或按 `Ctrl+Shift+X`）
3. 搜索 "Live Server"
4. 点击安装

### 使用Live Server
1. 在VS Code中打开项目文件夹
2. 右键点击 `prototype-index.html`
3. 选择 "Open with Live Server"
4. 浏览器会自动打开页面
5. 地址栏会显示类似 `http://127.0.0.1:5500/prototype-index.html` 的URL

## 方法四：使用PHP内置服务器（如果已安装PHP）

```bash
# 进入项目目录
cd "E:\STUDY\Works\毕设"

# 启动服务器
php -S localhost:8000
```

## 常见问题

### Q: Python命令不存在？
A: 
- Windows: 从 https://www.python.org/ 下载安装Python
- Mac: 通常已预装，尝试 `python3`
- Linux: 运行 `sudo apt install python3` (Ubuntu/Debian)

### Q: 端口8000被占用？
A: 使用其他端口，如：
```bash
python -m http.server 8080
```
然后访问 `http://localhost:8080/prototype-index.html`

### Q: 防火墙阻止访问？
A: 
- Windows: 允许Python通过防火墙
- Mac/Linux: 通常不需要额外设置

### Q: 如何停止服务器？
A: 在终端中按 `Ctrl + C`

## 推荐工作流程

1. **启动本地服务器**（使用Python最简单）
2. **在浏览器中打开** `http://localhost:8000/prototype-index.html`
3. **确认页面正常显示**
4. **打开Figma** → 安装 "html.to.design" 插件
5. **在插件中输入URL**：`http://localhost:8000/prototype-index.html`
6. **点击Import** → 等待转换完成

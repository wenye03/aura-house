# 启动本地服务器 - 详细说明

## 方法一：使用启动脚本（最简单）

### Windows用户：
1. **双击运行** `start-server.bat` 文件
2. 会打开一个黑色窗口，显示服务器信息
3. 在浏览器中访问：`http://localhost:8000/prototype-index.html`
4. 需要停止时，在黑色窗口中按 `Ctrl + C`

### 如果双击无法运行：
1. 右键点击 `start-server.bat`
2. 选择"以管理员身份运行"

## 方法二：手动启动（如果脚本不工作）

### 步骤：

1. **打开命令提示符（CMD）**
   - 按 `Win + R`
   - 输入 `cmd`
   - 按回车

2. **进入项目目录**
   ```cmd
   cd /d E:\STUDY\Works\毕设
   ```

3. **启动服务器**
   ```cmd
   python -m http.server 8000
   ```
   
   如果提示"python不是内部或外部命令"，尝试：
   ```cmd
   python3 -m http.server 8000
   ```
   或
   ```cmd
   py -m http.server 8000
   ```

4. **看到以下信息表示成功**：
   ```
   Serving HTTP on 0.0.0.0 port 8000 (http://0.0.0.0:8000/) ...
   ```

5. **在浏览器中访问**
   - 打开浏览器
   - 输入：`http://localhost:8000/prototype-index.html`
   - 或：`http://127.0.0.1:8000/prototype-index.html`

## 方法三：使用VS Code的Live Server（推荐）

### 如果安装了VS Code：

1. **打开VS Code**
2. **打开项目文件夹**（E:\STUDY\Works\毕设）
3. **安装Live Server插件**（如果还没安装）
   - 点击左侧扩展图标
   - 搜索 "Live Server"
   - 点击安装
4. **启动服务器**
   - 右键点击 `prototype-index.html`
   - 选择 "Open with Live Server"
   - 浏览器会自动打开

## 方法四：使用Node.js（如果已安装）

### 如果已安装Node.js：

1. **打开命令提示符**
2. **进入项目目录**
   ```cmd
   cd /d E:\STUDY\Works\毕设
   ```

3. **安装http-server**（只需安装一次）
   ```cmd
   npm install -g http-server
   ```

4. **启动服务器**
   ```cmd
   http-server -p 8000
   ```

## 常见问题解决

### Q1: 提示"python不是内部或外部命令"
**解决方法：**
1. 访问 https://www.python.org/downloads/
2. 下载并安装Python
3. 安装时勾选"Add Python to PATH"
4. 重启命令提示符后重试

### Q2: 端口8000被占用
**解决方法：**
使用其他端口，如8080：
```cmd
python -m http.server 8080
```
然后访问：`http://localhost:8080/prototype-index.html`

### Q3: 防火墙阻止访问
**解决方法：**
1. Windows会弹出防火墙提示，点击"允许访问"
2. 或在Windows防火墙设置中允许Python通过

### Q4: 浏览器显示"无法访问此网站"
**检查清单：**
- ✅ 服务器是否正在运行？（黑色窗口是否打开）
- ✅ 是否使用了正确的URL？（注意是localhost，不是127.0.0.1）
- ✅ 端口号是否正确？（默认8000）
- ✅ 文件名是否正确？（prototype-index.html）

### Q5: 页面显示但样式丢失
**原因：** 这是正常的，因为Tailwind CSS和Font Awesome是从CDN加载的
**解决方法：** 确保网络连接正常，CDN资源可以正常加载

## 验证服务器是否运行

在浏览器地址栏输入：
```
http://localhost:8000/
```

如果看到文件列表（包括prototype-index.html），说明服务器运行正常。

## 快速测试

1. 双击 `start-server.bat`
2. 打开浏览器
3. 访问 `http://localhost:8000/prototype-index.html`
4. 如果看到页面，说明成功！

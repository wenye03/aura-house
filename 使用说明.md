# 智能家居App - 高保真原型使用说明

## 📱 项目结构

```
智能家居App/
├── prototype-index.html  # 主入口文件（所有页面预览）
├── home.html            # 首页
├── rooms.html           # 房间页面
├── device.html          # 设备控制页面
├── scenes.html          # 场景页面
├── profile.html         # 个人中心页面
└── README.md            # 项目说明文档
```

## 🚀 使用方法

### 方式一：查看所有原型页面（推荐）
直接在浏览器中打开 `prototype-index.html` 文件，即可看到所有页面的平铺展示。

### 方式二：单独查看某个页面
直接打开对应的HTML文件，例如：
- `home.html` - 查看首页
- `rooms.html` - 查看房间页面
- `device.html` - 查看设备控制页面
- `scenes.html` - 查看场景页面
- `profile.html` - 查看个人中心页面

## 📋 功能说明

### 1. 首页 (home.html)
- 欢迎信息和环境数据展示（温度、湿度）
- 快捷设备控制（4个常用设备）
- 房间快捷入口
- 底部Tab导航栏

### 2. 房间页面 (rooms.html)
- 房间列表展示（客厅、卧室、厨房、浴室）
- 每个房间的设备概览
- 房间大图和设备状态
- 添加房间功能

### 3. 设备控制页面 (device.html)
- 设备大图和状态展示
- 电源开关控制
- 亮度调节滑块
- 色温调节
- 快捷场景按钮
- 定时设置
- 设备详细信息

### 4. 场景页面 (scenes.html)
- 场景卡片展示（回家、离家、睡眠、观影、阅读、聚会）
- 一键执行场景功能
- 添加新场景功能

### 5. 个人中心页面 (profile.html)
- 用户信息展示
- 统计数据（设备总数、房间数、场景数）
- 设置菜单（我的设备、家庭成员、消息通知等）
- 退出登录功能

## 🎨 设计特点

1. **iOS设计规范**
   - 遵循iOS Human Interface Guidelines
   - 使用SF Pro系统字体
   - iOS风格的状态栏和Tab导航栏

2. **iPhone 15 Pro模拟**
   - 393 × 852 pt 尺寸
   - 47px圆角边框
   - 动态岛设计

3. **真实图片**
   - 使用Unsplash高质量图片
   - 真实的家居场景照片

4. **现代化UI**
   - Tailwind CSS样式框架
   - Font Awesome图标库
   - 渐变背景和毛玻璃效果
   - 流畅的过渡动画

## 💡 技术栈

- **HTML5** - 页面结构
- **Tailwind CSS** - 样式框架（CDN）
- **Font Awesome 6** - 图标库（CDN）
- **Unsplash** - 真实图片资源

## 📝 注意事项

1. **网络连接**：需要网络连接以加载CDN资源（Tailwind CSS和Font Awesome）
2. **浏览器兼容性**：建议使用Chrome、Safari或Edge等现代浏览器
3. **index.html目录**：如果项目根目录存在`index.html`目录，请使用`prototype-index.html`作为主入口文件
